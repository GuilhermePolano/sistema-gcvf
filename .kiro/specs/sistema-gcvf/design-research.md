# Pesquisa de Design - Sistema FIERGS

## Identidade Visual FIERGS

### Cores Principais
Com base na identidade visual da FIERGS (Federação das Indústrias do Estado do Rio Grande do Sul):

**Cores Primárias:**
- **Azul FIERGS**: #003366 (azul escuro institucional)
- **Azul Secundário**: #0066CC (azul médio)
- **Azul Claro**: #E6F2FF (azul muito claro para backgrounds)

**Cores de Apoio:**
- **Cinza Escuro**: #333333 (textos principais)
- **Cinza Médio**: #666666 (textos secundários)
- **Cinza Claro**: #F5F5F5 (backgrounds neutros)
- **Branco**: #FFFFFF (backgrounds principais)

**Cores de Status:**
- **Verde Sucesso**: #28A745 (feedbacks positivos, conclusões)
- **Amarelo Atenção**: #FFC107 (alertas, pendências)
- **Vermelho Erro**: #DC3545 (erros, campos obrigatórios)
- **Laranja Info**: #FD7E14 (informações importantes)

### Tipografia
- **Fonte Principal**: Roboto ou similar (sans-serif moderna)
- **Fonte Secundária**: Open Sans (para textos longos)
- **Hierarquia**: H1-H6 com pesos 300, 400, 500, 700

### Elementos Visuais
- **Bordas**: Raio de 4px para elementos pequenos, 8px para cards
- **Sombras**: Sutis, com opacidade baixa
- **Espaçamento**: Grid de 8px (8, 16, 24, 32, 40px)
- **Ícones**: Outline style, 24px padrão

## Diretrizes de UX

### Princípios
1. **Clareza**: Interface limpa e objetiva
2. **Eficiência**: Fluxos otimizados para tarefas frequentes
3. **Consistência**: Padrões visuais uniformes
4. **Acessibilidade**: Contraste adequado e navegação por teclado
5. **Responsividade**: Adaptação para desktop, tablet e mobile

### Layout
- **Desktop**: Layout de 12 colunas, max-width 1200px
- **Tablet**: Layout de 8 colunas, adaptativo
- **Mobile**: Layout de 4 colunas, stack vertical

## Componentes Base Sugeridos

### Navegação
- **Header**: Logo FIERGS + menu principal + perfil do usuário
- **Sidebar**: Menu lateral retrátil com ícones e labels
- **Breadcrumb**: Navegação hierárquica

### Formulários
- **Input Fields**: Bordas sutis, focus em azul FIERGS
- **Buttons**: Primário (azul), secundário (outline), terciário (ghost)
- **Dropdowns**: Com busca para listas longas
- **Date Pickers**: Calendário integrado

### Dados
- **Cards**: Para exibir informações de funcionários/PDIs
- **Tables**: Responsivas com ordenação e filtros
- **Charts**: Para métricas e relatórios (Chart.js ou similar)
- **Progress Bars**: Para acompanhamento de PDIs

### Feedback
- **Modals**: Para confirmações e formulários complexos
- **Toasts**: Notificações não-intrusivas
- **Loading States**: Skeletons e spinners
- **Empty States**: Ilustrações e CTAs claros
